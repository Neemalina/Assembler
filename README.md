# Ассемблер для учебной виртуальной машины (УВМ)

Ассемблер для трансляции программ на языке ассемблера УВМ в промежуточное представление.

## Спецификация языка ассемблера

### Синтаксис
- Одна команда на строку
- Комментарии начинаются с символа `;`
- Аргументы разделяются запятыми
- Регистр мнемоник не важен
- Поддерживаются десятичные и шестнадцатеричные числа (с префиксом `0x`)

### Команды

#### 1. LOAD_CONST - Загрузка константы
Загружает константу в память по указанному адресу.

**Формат:** `LOAD_CONST адрес, константа`

**Поля команды:**
- `A` = 20 (код операции)
- `B` = адрес
- `C` = константа

**Пример:**
```asm
LOAD_CONST 811, 213
```
#### 2. STORE / ФАС - Запись значения в память
Копирует значение из одного адреса памяти в другой.

**Формат:** `STORE адрес_источник, адрес_назначение`  
**Альтернатива:** `ФАС адрес_источник, адрес_назначение`

**Поля команды:**
- `A` = 12 (код операции)
- `B` = адрес_источник
- `C` = адрес_назначение

**Размер:** 5 байт

**Пример:**
```asm
STORE 709, 447
ФАС 709, 447
```
#### 3. BINARY_OP / ФЭЗ - Бинарная операция

Выполняет бинарную операцию над двумя операндами.

### Поля команды:
- **A** = 19 (код операции)
- **B** = адрес_результата
